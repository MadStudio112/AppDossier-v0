03 — Komponenten

UI-Bibliothek

Die Applikation nutzt shadcn/ui als Komponentenbasis — gebaut auf Radix UI Primitives, gestylt mit Tailwind CSS. Über 40 Komponenten sind konfiguriert und anpassbar.

Input & Formulare

  • Button, Input, Textarea, Label
  • Select, Checkbox, Radio Group, Switch
  • Slider (Rating-Kriterien)
  • Dialog, Alert Dialog, Popover
  • Input OTP, Toggle, Toast
  • Accordion, Dropdown Menu

Layout & Navigation

  • Avatar, Badge, Separator
  • Tabs, Navigation Menu, Menubar
  • Scroll Area, Collapsible
  • Tooltip, Hover Card, Context Menu
  • Progress, Skeleton
  • Sidebar (Radix-basiert)

Daten & Visualisierung

  • TanStack Table — Sortierbare, filterbare Datentabellen
  • Recharts — Radar-Charts für 5-Kriterien-Bewertungen
  • Embla Carousel — Touch-fähige Karusselle
  • cmdk — Command Palette (Cmd+K)
  • Vaul — Mobile Drawer-Komponente
  • Sonner — Toast-Notifications

Custom Components

  • QCard — Match-Rating-Widget
  • StarRating — Rating-Input
  • AppLogo — SVG-Logo mit Varianten
  • AuthGuard — Rollen-basierter Schutz
  • LanguageSwitcher — DE/EN Toggle
  • CookieBanner — DSGVO-konform
04 — Brand

Markenidentität

Name & Logo

Der Name wird konsequent als TEAMCHEF geschrieben — in zwei Teilen:

TEAMCHEF

  • TEAM in Weiß — steht für Gemeinschaft & Community
  • CHEF in Grün — steht für Führung & Entscheidung
  • Tagline: „Deine Stimme. Dein Team. Dein Ranking."

Visuelles Konzept

  • Dark-first — Dunkle Backgrounds als Standard
  • Grün als Primärfarbe — Energie, Fußballrasen
  • Noise Texture — SVG Fractal Noise Overlay
  • Gradient Backgrounds — Subtile Radial-Verläufe
  • Card-based Layout — Informations-Hierarchie

Motion & Animation

  • Bewusst minimal — Performance vor Effekt
  • Pulse — Badge-Indikator Animation
  • Hover Transitions — 150–200ms, subtle
  • Framer Motion — Nur für Key-Interactions
  • Chart Animations — Radar-Chart Reveals
